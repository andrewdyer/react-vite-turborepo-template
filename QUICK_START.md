# react-vite-turborepo-template

This project was created from the **[React + Vite + Turborepo Template](https://github.com/andrewdyer/react-vite-turborepo-template)**.

## Getting Started

Before working on the codebase, read the **[contributing guide](./CONTRIBUTING.md)**.

### Install

Install the dependencies:

```bash
pnpm install
```

### Build

Build the shared packages:

```bash
pnpm build
```

### Start

Start the development server:

```bash
pnpm dev
```
